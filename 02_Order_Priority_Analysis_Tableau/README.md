This Tableau project analyzes sales, profit, and monthly trends by order priority. The dashboard allows users to filter by order priority and year to better understand how different order priority groups contribute to sales performance over time.

> **Data Note:** This project uses sample data for portfolio demonstration purposes. It does not contain private client data or real customer personal data.

> **Workbook Note:** The Tableau workbook file is not included for this project. This portfolio entry documents the dashboard using exported screenshots only.

## Business Problem

Business users need to understand how order priority impacts sales, profit, and category performance. Different priority levels may show different patterns in order value, profitability, and seasonality.

This dashboard answers questions such as:

* Which order priority groups drive the most sales?
* How do sales and profit vary by order priority?
* Which product categories perform best within each priority group?
* How do monthly sales trends change when filtering by priority and year?
* Are high priority orders consistently more valuable or profitable?

---

## Dashboard Features

| Feature | Description |
|---|---|
| Order priority filter | Allows users to focus on Critical, High, Medium, or Low priority orders |
| Year filter | Allows users to analyze order priority performance by year |
| Sales and profit scatter plot | Shows relationship between sales and profit per order |
| Sales by order priority | Compares category sales across priority groups |
| Monthly sales trend | Shows how sales change over time |
| Interactive highlighting | Lets users select a priority group from the legend to highlight related marks |
